## ℹ️ Description | تفصیل

An advanced offline-first single-file search engine application designed for in-depth research within the Maktaba Jibreel collection. It empowers users with powerful regular expression (regex) search capabilities, synonym matching, and various filtering options to efficiently navigate through a vast corpus of Islamic texts, including Fatawa, Fiqh, Quranic tafsir, Lughat (dictionaries), and more. The app is built to be robust, error-free, and prioritizes offline accessibility after an initial data load.

مکتبہ جبریل کے مجموعے میں گہرائی سے تحقیق کے لیے ڈیزائن کیا گیا ایک جدید آف لائن-فرسٹ سنگل فائل سرچ انجن ایپلی کیشن۔ یہ صارفین کو طاقتور ریگولر ایکسپریشن (regex) تلاش کی صلاحیتوں، مترادف الفاظ کی مطابقت، اور اسلامی متون کے ایک وسیع مجموعے، بشمول فتاویٰ، فقہ، قرآنی تفسیر، لغات، اور مزید بہت کچھ، میں مؤثر طریقے سے نیویگیٹ کرنے کے لیے مختلف فلٹرنگ آپشنز فراہم کرتا ہے۔ ایپ کو مضبوط، غلطی سے پاک بنایا گیا ہے، اور ابتدائی ڈیٹا لوڈ کے بعد آف لائن رسائی کو ترجیح دیتی ہے۔

## ✨ Features | اہم خصوصیات

*   **🔍 Advanced Search with Regular Expressions (Regex)** | **ریگولر ایکسپریشن (Regex) کے ساتھ جدید تلاش:** Perform complex and precise searches using regex patterns. | ریجیکس پیٹرن کا استعمال کرتے ہوئے پیچیدہ اور عین مطابق تلاش کریں۔
*   **🔄 Synonym Search (Urdu)** | **مترادف الفاظ کی تلاش (اردو):** Automatically include synonyms for Urdu search terms to broaden results. | نتائج کو وسیع کرنے کے لیے اردو تلاش کی اصطلاحات کے مترادفات خود بخود شامل کریں۔
*   **🇬🇧 Optional English Search** | **اختیاری انگریزی تلاش:** Toggle to search using English terms if needed. | اگر ضرورت ہو تو انگریزی اصطلاحات کا استعمال کرتے ہوئے تلاش کرنے کے لیے ٹوگل کریں۔
*   **✒️ Ignore Diacritics (اعراب) in Search** | **تلاش میں اعراب کو نظر انداز کریں:** Option to disregard Arabic/Urdu diacritical marks for more flexible matching. | زیادہ لچکدار میچنگ کے لیے عربی/اردو اعرابی علامات کو نظر انداز کرنے کا آپشن۔
*   **📚 Multiple Data Scopes** | **متعدد ڈیٹا کے دائرہ کار:** Search within specific categories like Quran Tafsir, Fatawa & Fiqh, Lughat, Funoon, Aqaid, Seerat, etc. | قرآن تفسیر، فتاویٰ و فقہ، لغات، فنون، عقائد، سیرت وغیرہ جیسی مخصوص کیٹیگریز میں تلاش کریں۔
*   **💾 Offline First Functionality** | **آف لائن فرسٹ فعالیت:** Works completely offline after the initial data download, thanks to `AppCache` and `localForage`. | `AppCache` اور `localForage` کی بدولت ابتدائی ڈیٹا ڈاؤن لوڈ کے بعد مکمل طور پر آف لائن کام کرتا ہے۔
*   **⚡ Fast and Responsive UI** | **تیز اور ریسپانسیو یوزر انٹرفیس:** Optimized for speed and a smooth user experience. | رفتار اور ہموار صارف کے تجربے کے لیے بہتر بنایا گیا ہے۔
*   **📜 Saved Search History & Context Menu** | **محفوظ شدہ تلاش کی سرگزشت اور کانٹیکسٹ مینیو:** Quickly access previous searches. Right-click (or long-press on mobile) on search history items to remove them. | پچھلی تلاشوں تک تیزی سے رسائی حاصل کریں۔ تلاش کی سرگزشت کے آئٹمز کو ہٹانے کے لیے ان پر دائیں کلک کریں (یا موبائل پر دیر تک دبائیں)۔
*   **📄 Filter Search Results** | **تلاش کے نتائج کو فلٹر کریں:** Further refine your search results with an additional filter input. | اضافی فلٹر ان پٹ کے ساتھ اپنے تلاش کے نتائج کو مزید بہتر بنائیں۔
*   **📱 Mobile Responsive Design** | **موبائل ریسپانسیو ڈیزائن:** Adapts gracefully to various screen sizes. | مختلف اسکرین سائزز کے مطابق احسن طریقے سے ڈھل جاتا ہے۔
*   **🔄 Data Management (Backup/Restore)** | **ڈیٹا کا انتظام (بیک اپ/بحالی):**

## 🚀 How to Use | استعمال کرنے کا طریقہ

1.  **📥 Download/Clone the Repository** | **ریپوزٹری ڈاؤن لوڈ/کلون کریں:** Get the files onto your local system. | فائلیں اپنے مقامی سسٹم پر حاصل کریں۔
2.  **📂 Open `index.html`** | **`index.html` کھولیں:** Open the `index.html` file in a modern web browser (Chrome, Firefox recommended). | `index.html` فائل کو جدید ویب براؤزر (کروم، فائر فاکس تجویز کردہ) میں کھولیں۔
3.  **⏳ Initial Data Load** | **ابتدائی ڈیٹا لوڈ:** On the first launch, the application will download and cache the necessary data. Progress will be displayed. This only happens once or when data needs updating. | پہلی بار لانچ کرنے پر، ایپلیکیشن ضروری ڈیٹا ڈاؤن لوڈ اور کیش کرے گی۔ پیشرفت دکھائی جائے گی۔ یہ صرف ایک بار ہوتا ہے یا جب ڈیٹا کو اپ ڈیٹ کرنے کی ضرورت ہوتی ہے۔
4.  **⌨️ Start Searching** | **تلاش شروع کریں:**
    *   Enter your query in the main search box (e.g., `نماز کس طرح`). | مرکزی سرچ باکس میں اپنی مطلوبہ عبارت درج کریں (مثلاً `نماز کس طرح`)۔
    *   Use the checkboxes under "Options" to toggle features like "مترادف الفاظ" (Synonyms), "اعراب کو نظر انداز" (Ignore Diacritics), etc. | "آپشنز" کے تحت چیک باکسز کا استعمال کرتے ہوئے خصوصیات جیسے "مترادف الفاظ"، "اعراب کو نظر انداز" وغیرہ کو ٹوگل کریں۔
    *   If available, select a specific data scope (e.g., "فتاوی فقہ", "قرآن تفسیر") from the "تلاش کا دائرہ منتخب کریں" section. | اگر دستیاب ہو تو، "تلاش کا دائرہ منتخب کریں" سیکشن سے ایک مخصوص ڈیٹا کا دائرہ (مثلاً "فتاوی فقہ"، "قرآن تفسیر") منتخب کریں۔
5.  **💡 Advanced Search Syntax | جدید تلاش کا نحو:**
    *   `لفظ1 لفظ2`: Finds items containing "لفظ1" followed by "لفظ2". | "لفظ1" کے بعد "لفظ2" پر مشتمل آئٹمز تلاش کرتا ہے۔
    *   `لفظ1|لفظ2`: Finds items containing "لفظ1" OR "لفظ2". | "لفظ1" یا "لفظ2" پر مشتمل آئٹمز تلاش کرتا ہے۔
    *   `لفظ1&لفظ2`: Finds items containing both "لفظ1" AND "لفظ2" (order doesn't matter). | "لفظ1" اور "لفظ2" دونوں پر مشتمل آئٹمز تلاش کرتا ہے (ترتیب اہم نہیں)۔
    *   `لفظ%`: Matches "لفظ" as a whole word. | "لفظ" کو مکمل لفظ کے طور پر ملاتا ہے۔
    *   `عبارت.*دوسری`: Matches "عبارت" followed by any characters and then "دوسری". | "عبارت" کے بعد کوئی بھی حروف اور پھر "دوسری" سے ملاتا ہے۔
    *   Use `۔` for "any character, zero or more times" (`.{0,}`), `،` for "any character, one or more times" (`.+`), `’` for "previous character is optional" (`.?`). | `۔` کا استعمال کریں "کوئی بھی حرف، صفر یا زیادہ بار" (`.{0,}`), `،` "کوئی بھی حرف، ایک یا زیادہ بار" (`.+`), `’` "پچھلا حرف اختیاری ہے" (`.?`) کے لیے۔
    *   Refer to the app's help section (if available) or standard regex guides for more. | مزید کے لیے ایپ کے ہیلپ سیکشن (اگر دستیاب ہو) یا معیاری ریجیکس گائیڈز سے رجوع کریں۔
6.  **📊 Filter Results** | **نتائج کو فلٹر کریں:** After an initial search, use the "Filter تلاش کو فلٹر کریں" box to further narrow down the displayed results. | ابتدائی تلاش کے بعد، دکھائے گئے نتائج کو مزید محدود کرنے کے لیے "Filter تلاش کو فلٹر کریں" باکس کا استعمال کریں۔
7.  **🗑️ Manage Search History** | **تلاش کی سرگزشت کا انتظام کریں:**
    *   Previously searched terms appear below the input or in the context menu. | پہلے تلاش کی گئی اصطلاحات ان پٹ کے نیچے یا کانٹیکسٹ مینیو میں ظاہر ہوتی ہیں۔
    *   Click a term to search it again. | اسے دوبارہ تلاش کرنے کے لیے کسی اصطلاح پر کلک کریں۔
    *   Right-click (or long-press) on a term in the history to remove it. | تاریخ میں کسی اصطلاح کو ہٹانے کے لیے اس پر دائیں کلک کریں (یا دیر تک دبائیں)۔
